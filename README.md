# Backend-Bolierplate


## Description

[Boilerplate](https://github.com) backend-boilerplate repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# serverless
$ npm run start:sls

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Linting and formatting

```bash
# Run lint
$ npm run lint

# Code formatting
$ npm run format
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
